#MyClassPHP
MyClassPHP是一款学习型框架。可以帮助初学者更快的了解框架原理。以及OOP工作原理
官方交流群，提交BUG群  438695935
##主要功能
1.路由模式  
    1>普通模式 index.php?c=Index&a=index  
    2>路径模式 index.php/Index/Index  
2.控制器  
    1>C()实例化控制器  
    2>display()载入模板 写入数据等  
3.模型  
    1>模型创建，增删改查等  
4.时间  
    1>获取时间  
    2>设置时间  
5.验证码  
    1>生成验证码  
6.验证  
    1>只需简单两步即可对数据进行验证  
7.上传  
    1>支持单文件多文件上传。自定义配置  
等一系列功能
